# t5-summarization
Summarization model using T5-base for Korean

## Requirements
- torch==1.6.0
- transformers==3.0.1
- streamlit==0.62.1
- gdown

## How to use
1. install requirements
```
pip install -r requirement.txt
```

2. Download model binary
```
python download.py
```
- model binary : 800MB

3. Run streamlit (default port:8501)
```
run.sh
```
